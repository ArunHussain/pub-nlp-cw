Repo for NLP coursework.
Model-related code is in 'BestModel/'.
The trained best model is not there due to size but would exist at 'BestModel/best_model.pt' if the training is run.

Run training with: 'uv run BestModel/train.py'
Run evaluation with: 'uv run evaluation.py', after training, to ensure 'best_model.pt' exists.
Run data exploration with: 'uv run data-exploration.py'.
